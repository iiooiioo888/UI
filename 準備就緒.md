## 🚀 立即開始 Figma 設計

### 第一步：打開 Figma（30 秒）
```
1. 訪問：https://figma.com
2. 登入或註冊（免費）
```

### 第二步：創建新文件（1 分鐘）
```
1. 點擊「+」→「Design File」
2. 命名：「年結表計算器 UI 設計」
```

### 第三步：打開參考文檔
```
主要文檔：
- skills/figma-design-helper/GUIDE.md（逐步操作指南）
- skills/figma-design-helper/REFERENCE.md（速查參考）

輔助文檔：
- UI/docs/Figma 設計逐步操作指南.md（詳細版）
- UI/docs/Figma 設計實戰工作手冊.md（問題解答）
```

### 第四步：開始設計（4 小時）
```
按照 GUIDE.md 的七個步驟：
1. 創建頁面結構（5 分鐘）
2. 創建設計系統（30 分鐘）
3. 創建組件庫（60 分鐘）
4. 設計頁面（90 分鐘）
```
