## 🎨 設計交付物

### 1. 頁面設計稿
- 首頁
- 查詢輸入頁（核心頁面）
- 結果展示頁
- 管理後台頁（精算管理員）
- 系統管理頁（IT 管理員）

### 2. 組件庫
- 按鈕（主按鈕/次要按鈕/危險按鈕/鏈接按鈕）
- 輸入框（默認/聚焦/錯誤/禁用）
- 單選按鈕組
- 日期選擇器
- 卡片
- 表格
- 彈窗
- 通知提示

### 3. 交互原型
- 查詢流程（輸入 → 驗證 → 計算 → 結果）
- 表單驗證（錯誤提示、即時反饋）
- 加載狀態（計算中、提交中）

### 4. 設計規範文檔
- 顏色規範
- 字體規範
- 間距規範
- 交互規範
