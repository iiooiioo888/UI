### 交互細節

#### 快速查詢卡片點擊
```
用戶動作：點擊「立即查詢」或卡片任意位置
系統反應：
  - 卡片有 Hover 效果（陰影加深）
  - 頁面跳轉至查詢輸入頁
  - 動畫：從右側滑入（300ms）
```

#### 最近查詢記錄點擊
```
用戶動作：點擊「查看」按鈕
系統反應：
  - 頁面跳轉至結果展示頁
  - 加載該次查詢的結果
  - 動畫：從右側滑入（300ms）
```

#### 查看全部點擊
```
用戶動作：點擊「查看全部」
系統反應：
  - 跳轉至查詢記錄列表頁（如有）
  - 或彈窗顯示所有記錄
```

### 交互細節

#### 導出 PDF

```
用戶動作：點擊「📄 導出 PDF」按鈕
系統反應：
  - 按鈕變為加載狀態
  - 文字：「生成中...」
  - Spinner 旋轉
  - 生成 PDF 文件
  - 自動下載到本地
  - 下載完成後顯示通知：「PDF 已生成並下載」
  - 按鈕恢復正常狀態
```

**PDF 格式**：
- 頁面大小：A4
- 方向：縱向
- 頁邊距：2.54cm
- 包含：所有表格和說明
- 頁腳：頁碼 + 生成日期
