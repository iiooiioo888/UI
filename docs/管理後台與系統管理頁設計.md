##### 導出產品列表

```
用戶動作：點擊「📥 導出產品列表」按鈕
系統反應：
  - 生成 Excel 文件
  - 自動下載
  - 顯示通知：「產品列表已導出」
```

**Excel 格式**：
- 包含所有字段
- 包含所有產品（不分頁）
- 文件名：`產品列表_YYYYMMDD.xlsx`

---

##### 導入產品

```
用戶動作：點擊「📤 導入產品」按鈕
系統反應：
  - 顯示文件選擇對話框
  - 僅接受 .xlsx 文件
```

**導入流程**：
```
1. 選擇文件
2. 系統驗證格式
3. 顯示預覽（前 10 行）
4. 顯示驗證結果
5. 確認導入
6. 顯示導入結果
```

**驗證規則**：
- 必填字段檢查
- 格式檢查
- 重複代碼檢查
