## ✨ 交互細節

### 1. 表單驗證

#### 必填字段驗證

**觸發時機**：失去焦點時

**錯誤樣式**：
```
┌─────────────────────────────┐
│  2026-13-45                 │  ← 紅色邊框 #f5222d
└─────────────────────────────┘
   ⚠️ 日期格式無效             ← 紅色文字 #f5222d，12px
```

**驗證規則**：
| 字段 | 規則 | 錯誤提示 |
|------|------|----------|
| 退保日期 | 有效日期 | 「日期格式無效」 |
| 繳至日期 | 有效日期 | 「日期格式無效」 |
| 產品名稱 | 已選擇 | 「請選擇產品」 |
| 保單生效日 | 有效日期 | 「日期格式無效」 |
| 年齡 | 0-120 | 「年齡必須在 0-120 之間」 |
| 性別 | 已選擇 | 「請選擇性別」 |
| 吸煙狀態 | 已選擇 | 「請選擇吸煙狀態」 |
| 投保金額 | >0 | 「投保金額必須大於 0」 |
| 已繳保費 | ≥0 | 「請輸入有效數字」 |

---

#### 字段聯動驗證

**規則 1**：繳至日期 ≥ 保單生效日
```
錯誤提示：「繳至日期必須晚於保單生效日」
```

**規則 2**：退保日期 ≥ 保單生效日
```
錯誤提示：「退保日期必須晚於保單生效日」
```

**規則 3**：年齡與保單生效日匹配
```
錯誤提示：「年齡與保單生效日不匹配」
```

---

### 2. 智能建議

#### 現金儲值結餘自動建議

當用戶輸入「現金儲值結餘」時：

1. 系統自動計算建議值
2. 顯示在輸入框右側

```
┌─────────────────────────────┐
│  123,456.78   [使用建議值]  │
└─────────────────────────────┘
```

3. 點擊「使用建議值」自動填充

---

#### 日期自動填充

當用戶輸入「紅利結餘日期」時：

1. 默認建議為退保日期
2. 可手動修改

---

### 3. 計算流程

#### 步驟 1：點擊「計算」按鈕

```
[計算] → [⟳ 計算中...]  ← 按鈕禁用
```

**按鈕狀態變化**：
- 文字變為「計算中...」
- 左側顯示 spinner
- 按鈕禁用（防止重複提交）

---

#### 步驟 2：加載提示（超過 2 秒）

如果計算超過 2 秒，顯示彈窗：

```
┌─────────────────────────────────────┐
│                                     │
│              ⟳                     │
│         計算中，請稍候...            │
│         正在計算 350 個欄位           │
│                                     │
└─────────────────────────────────────┘
```

---

#### 步驟 3a：計算成功

**提示**：右上角綠色通知
```
┌─────────────────────────────────────┐
│  ✅  計算成功                        │
└─────────────────────────────────────┘
```

**跳轉**：1 秒後自動跳轉至結果展示頁

---

#### 步驟 3b：計算失敗

**彈窗**：
```
┌─────────────────────────────────────┐
│  ⚠️  計算失敗                        │
├─────────────────────────────────────┤
│                                     │
│  所選產品的年齡無效。                │
│  請確認年齡在 18-80 歲之間。          │
│                                     │
│                  [確定]             │
│                                     │
└─────────────────────────────────────┘
```

**按鈕恢復**：
- 文字恢復為「計算」
- Spinner 消失
- 按鈕啟用

---

### 4. 表單重置

**點擊「重置」按鈕**：

1. 清空所有輸入字段
2. 清除所有錯誤提示
3. 產品選擇恢復默認（第一個）
4. 「其他」區域恢復收起狀態

**確認彈窗**（如果有已輸入內容）：
```
┌─────────────────────────────────────┐
│  確認重置？                          │
├─────────────────────────────────────┤
│  所有已輸入的內容將被清空。          │
│                                     │
│           [取消]  [確認重置]        │
└─────────────────────────────────────┘
```

## 📱 響應式適配

### 桌面端（≥1440px）

- 標準三列佈局
- 卡片寬度：`100%`
- 輸入框三列排列

### 平板端（1024px-1439px）

- 卡片寬度：`100%`
- 輸入框兩列排列
- 單選按鈕組換行

### 小屏幕（<1024px）

- 卡片寬度：`100%`
- 輸入框單列排列
- 表格橫向滾動

## 🎯 無障礙設計

### 鍵盤導航

| 按鍵 | 功能 |
|------|------|
| `Tab` | 下一字段 |
| `Shift+Tab` | 上一字段 |
| `Enter` | 提交表單 |
| `Esc` | 關閉彈窗 |
| `Space` | 切換單選按鈕 |

### 屏幕閱讀器

- 所有輸入框有 `label` 關聯
- 錯誤提示與輸入框關聯
- 必填字段標記 `*` 有 `aria-label`
