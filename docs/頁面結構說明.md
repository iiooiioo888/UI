# 頁面結構說明

## 頁面清單

| 頁面名稱 | 路徑 | 訪問權限 | 說明 |
|----------|------|----------|------|
| 首頁 | `/` | 所有用戶 | 系統入口，快速導航 |
| 查詢輸入頁 | `/query/input` | 所有用戶 | 輸入保單資料並計算 |
| 結果展示頁 | `/query/result` | 所有用戶 | 展示年結表結果 |
| 管理後台頁 | `/admin/data` | 精算管理員 | 費率表管理、產品配置 |
| 系統管理頁 | `/admin/system` | IT 管理員 | 用戶管理、系統設置 |
| 查詢記錄頁 | `/history` | 所有用戶 | 查看個人查詢歷史 |
| 變更歷史頁 | `/admin/changelog` | 精算管理員 | 費率表變更記錄 |

---

## 頁面詳細說明

### 1. 首頁 (`/`)

**組件結構**：
```
首頁
├── 頂部導航欄
│   ├── Logo + 系統名稱
│   ├── 用戶信息（頭像 + 姓名）
│   └── 退出登錄按鈕
├── 快速入口卡片區
│   ├── 快速查詢（大按鈕）
│   ├── 管理後台（僅精算管理員可見）
│   └── 系統管理（僅 IT 管理員可見）
├── 最近查詢記錄（列表）
│   └── 最多顯示 5 條
└── 系統公告區
    └── 精算部發布的更新通知
```

**業務邏輯**：
- 加載用戶最近 5 條查詢記錄
- 加載系統公告（如有）
- 根據用戶角色顯示/隱藏管理入口

---

### 2. 查詢輸入頁 (`/query/input`)

**組件結構**：
```
查詢輸入頁
├── 頂部導航欄（含返回按鈕）
├── 表單區域
│   ├── 【1. 保單資料】卡片
│   │   ├── 退保日期（日期選擇器）*
│   │   ├── 繳至日期（日期選擇器）*
│   │   ├── 產品名稱（單選按鈕組）*
│   │   ├── 保單生效日（日期選擇器）*
│   │   ├── 年齡（數字輸入框）*
│   │   ├── 性別（單選按鈕組）*
│   │   ├── 吸煙狀態（單選按鈕組）*
│   │   ├── 投保金額（數字輸入框）*
│   │   ├── 幣種（單選按鈕組）
│   │   └── 繳費方式（單選按鈕組）
│   ├── 【2. 賬戶價值】卡片
│   │   ├── 紅利結餘（數字輸入框）
│   │   ├── 現金儲值結餘（數字輸入框）
│   │   ├── 貸款結餘（數字輸入框）
│   │   └── 已繳保費（數字輸入框）*
│   ├── 【3. 其他】卡片（可折疊）
│   │   ├── 附加險產品代碼
│   │   ├── 附加險保額
│   │   ├── 紅利提取記錄
│   │   ├── 現金儲值提取記錄
│   │   ├── 員工折扣類型
│   │   └── 加費信息
│   └── 表單操作區
│       ├── 重置按鈕
│       └── 計算按鈕（提交）
└── 底部說明區
    └── 必填字段說明、幫助鏈接
```

**驗證邏輯**：
```javascript
// 必填字段驗證
if (!surrenderDate) { showError('退保日期為必填'); return false; }

// 年齡範圍驗證
if (age < 0 || age > 120) { showError('年齡必須在 0-120 之間'); return false; }

// 日期格式驗證
if (!isValidDate(policyDate)) { showError('保單生效日格式無效'); return false; }

// 產品有效性驗證
if (!productList.includes(selectedProduct)) { showError('所選產品無效'); return false; }
```

**交互流程**：
```
用戶輸入 → 失去焦點時驗證 → 顯示錯誤/通過
    ↓
點擊計算 → 全部驗證通過 → 調用計算 API
    ↓
計算成功 → 跳轉至結果展示頁
計算失敗 → 彈窗顯示錯誤原因
```

### 3. 結果展示頁 (`/query/result`)

**組件結構**：
```
結果展示頁
├── 頂部導航欄（含返回按鈕）
├── 操作按鈕區
│   ├── 導出 PDF
│   ├── 列印
│   ├── 發送郵件
│   └── 重新計算
├── 結果區域
│   ├── 【保單資料】表格
│   ├── 【保單戶口價值】表格
│   ├── 【說明摘要】列表
│   └── 【詳細計算結果】可展開區域
└── 底部操作區
    └── 返回查詢/重新計算
```

**數據展示**：
- 表格形式，帶邊框和交替行顏色
- 金額右對齊，帶千分位分隔符
- 支持中英文切換

**導出功能**：
- PDF：調用活字格 PDF 生成插件
- 列印：瀏覽器原生列印
- 郵件：彈窗輸入郵箱地址，調用 SMTP 發送

### 4. 管理後台頁 (`/admin/data`)

**組件結構**：
```
管理後台頁
├── 頂部導航欄
├── 功能選項卡
│   ├── 產品配置
│   ├── 費率表管理
│   └── 變更歷史
├── 產品配置選項卡
│   ├── 產品列表（表格）
│   │   ├── 代碼
│   │   ├── 產品名稱
│   │   ├── 狀態（在售/停售）
│   │   └── 操作（編輯/停用）
│   └── 新增產品按鈕
├── 費率表管理選項卡
│   ├── 當前版本信息
│   ├── 導出模板按鈕
│   ├── 上傳文件按鈕
│   └── 費率表預覽（表格）
└── 變更歷史選項卡
    └── 變更記錄列表（時間、用戶、內容）
```

**Excel 導入流程**：
```
1. 點擊「導出模板」→ 下載當前費率表 Excel
2. 精算部同事更新數據
3. 點擊「上傳文件」→ 選擇 Excel 文件
4. 系統驗證格式 → 顯示預覽
5. 確認無誤 → 提交
6. 生成新版本 → 記錄變更日誌
```

## 頁面跳轉關係

```
首頁
├── 點擊「快速查詢」→ 查詢輸入頁
├── 點擊「管理後台」→ 管理後台頁
└── 點擊「系統管理」→ 系統管理頁

查詢輸入頁
├── 計算成功 → 結果展示頁
└── 點擊返回 → 首頁

結果展示頁
├── 點擊「重新計算」→ 查詢輸入頁
└── 點擊返回 → 首頁

管理後台頁
└── 點擊返回 → 首頁

系統管理頁
└── 點擊返回 → 首頁
```
