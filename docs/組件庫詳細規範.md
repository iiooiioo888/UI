# 組件庫詳細規範

> 本文檔定義所有 UI 組件的詳細規格，供設計師和開發人員使用。

---

## 1. 按鈕 (Button)

### 1.1 主按鈕 (Primary Button)

**使用場景**：計算、提交、保存、確認

**樣式規格**：
```
┌─────────────────┐
│   計算          │
└─────────────────┘
```

| 屬性 | 值 |
|------|------|
| 背景色 | `#1890ff` |
| 文字顏色 | `#ffffff` |
| 邊框 | 無 |
| 圓角 | `4px` |
| 高度 | `32px` (默認) / `40px` (大) |
| 內邊距 | `8px 16px` |
| 字體大小 | `14px` |
| 字體粗細 | `400` |

**狀態**：
| 狀態 | 背景色 | 說明 |
|------|--------|------|
| 默認 | `#1890ff` | 正常狀態 |
| Hover | `#40a9ff` | 鼠標懸停 |
| Active | `#096dd9` | 點擊中 |
| Disabled | `#d9d9d9` | 禁用（文字 `#ffffff80`） |
| Loading | `#1890ff` | 加載中（顯示 spinner） |

---

### 1.2 次要按鈕 (Secondary Button)

**使用場景**：重置、取消、返回、上一步

**樣式規格**：
```
┌─────────────────┐
│   重置          │
└─────────────────┘
```

| 屬性 | 值 |
|------|------|
| 背景色 | `#ffffff` |
| 文字顏色 | `#595959` |
| 邊框 | `1px solid #d9d9d9` |
| 圓角 | `4px` |
| 高度 | `32px` |
| 內邊距 | `8px 16px` |

**狀態**：
| 狀態 | 背景色 | 邊框色 | 文字色 |
|------|--------|--------|--------|
| 默認 | `#ffffff` | `#d9d9d9` | `#595959` |
| Hover | `#ffffff` | `#1890ff` | `#1890ff` |
| Active | `#ffffff` | `#096dd9` | `#096dd9` |
| Disabled | `#fafafa` | `#d9d9d9` | `#bfbfbf` |

---

### 1.3 危險按鈕 (Danger Button)

**使用場景**：刪除、停用、清除所有

**狀態**：
| 狀態 | 背景色 |
|------|--------|
| 默認 | `#f5222d` |
| Hover | `#ff4d4f` |

| Disabled | `#d9d9d9` |

---

### 1.4 鏈接按鈕 (Link Button)

**使用場景**：展開詳情、查看更多、忘記密碼

**樣式規格**：
```
  展開詳情 ›
```

| 屬性 | 值 |
|------|------|
| 背景色 | 無 |
| 文字顏色 | `#1890ff` |
| 邊框 | 無 |
| 下劃線 | Hover 時顯示 |

---

### 1.5 按鈕尺寸

| 尺寸 | 高度 | 內邊距 | 字體大小 |
|------|------|--------|----------|
| 小 | `24px` | `4px 12px` | `12px` |
| 默認 | `32px` | `8px 16px` | `14px` |
| 大 | `40px` | `12px 24px` | `16px` |

## 2. 輸入框 (Input)

### 2.1 文本輸入框

**樣式規格**：
```
┌─────────────────────────────┐
│  請輸入...                   │
└─────────────────────────────┘
```

| 屬性 | 值 |
|------|------|
| 背景色 | `#ffffff` |
| 邊框色 | `#d9d9d9` |
| 邊框寬度 | `1px` |
| 圓角 | `4px` |
| 高度 | `32px` |
| 內邊距 | `8px 12px` |
| 字體大小 | `14px` |

**狀態**：
| 狀態 | 邊框色 | 說明 |
|------|--------|------|
| 默認 | `#d9d9d9` | 正常狀態 |
| Hover | `#40a9ff` | 鼠標懸停 |
| Focus | `#1890ff` | 輸入中（帶陰影） |
| Error | `#f5222d` | 驗證失敗 |
| Disabled | `#d9d9d9` | 禁用（背景 `#f5f5f5`） |

---

### 2.2 數字輸入框

**樣式規格**：同文本輸入框，右側帶上下箭頭

```
┌─────────────────────────────┐
│  123                    ▲   │
│                       ▼     │
└─────────────────────────────┘
```

**特殊屬性**：
- 文字右對齊
- 顯示千分位分隔符
- 可配置最小值/最大值

---

### 2.3 日期輸入框

**樣式規格**：
```
┌─────────────────────────────┐
│  📅 2026-03-24              │
└─────────────────────────────┘
```

**特殊屬性**：
- 左側顯示日曆圖標
- 點擊彈出日期選擇器
- 格式：`YYYY-MM-DD`

---

### 2.4 帶單位輸入框

**樣式規格**：
```
┌─────────────────────────────┐
│  123,456.78            HKD  │
└─────────────────────────────┘
```

**特殊屬性**：
- 右側顯示單位（HKD/USD/歲/%）
- 單位背景色：`#fafafa`
- 單位文字色：`#595959`

---

### 2.5 輸入框尺寸

| 尺寸 | 高度 | 適用場景 |
|------|------|----------|
| 小 | `24px` | 緊湊表單 |
| 默認 | `32px` | 常規表單 |
| 大 | `40px` | 重要輸入 |

---

## 5. 表格 (Table)

### 5.1 標準表格

**樣式規格**：
```
┌─────────────┬─────────────┬─────────────┐
│  項目       │  金額       │  日期       │
├─────────────┼─────────────┼─────────────┤
│  現金價值   │  123,456.78 │  2026-03-24 │
│  紅利結餘   │   12,345.67 │  2026-03-24 │
└─────────────┴─────────────┴─────────────┘
```

| 屬性 | 值 |
|------|------|
| 表頭背景 | `#fafafa` |
| 表頭文字 | `#262626`，粗體 |
| 行背景 | 交替 `#ffffff` / `#fafafa` |
| 邊框色 | `#d9d9d9` |
| 單元格內邊距 | `12px 16px` |
| 行高 | `40px` |

**交互**：
| 狀態 | 背景色 |
|------|--------|
| 默認 | 交替色 |
| Hover | `#e6f7ff` |
| Selected | `#bae7ff` |

---

### 5.2 金額列特殊格式

- 右對齊
- 千分位分隔符
- 小數點後 2 位
- 幣種符號在前（HKD / USD）

## 6. 彈窗 (Modal)

### 6.1 確認彈窗

**樣式規格**：
```
┌─────────────────────────────────────┐
│  確認刪除                     [✕]   │
├─────────────────────────────────────┤
│                                     │
│  確定要刪除此項嗎？此操作不可恢復。  │
│                                     │
│           [取消]  [確認刪除]        │
│                                     │
└─────────────────────────────────────┘
```

| 屬性 | 值 |
|------|------|
| 最小寬度 | `400px` |
| 背景色 | `#ffffff` |
| 圓角 | `8px` |
| 陰影 | `0 4px 12px rgba(0,0,0,0.15)` |
| 遮罩色 | `rgba(0,0,0,0.45)` |

---

### 6.2 錯誤彈窗

**樣式規格**：
```
┌─────────────────────────────────────┐
│  ⚠️  錯誤                            │
├─────────────────────────────────────┤
│                                     │
│  計算失敗：所選產品的年齡無效。      │
│  請確認年齡在 18-80 歲之間。          │
│                                     │
│                  [確定]             │
│                                     │
└─────────────────────────────────────┘
```

**特殊屬性**：
- 標題左側顯示警告圖標（紅色）
- 標題文字：`#f5222d`

---

### 6.3 成功彈窗

**樣式規格**：
```
┌─────────────────────────────────────┐
│  ✅  計算完成                        │
├─────────────────────────────────────┤
│                                     │
│  年結表計算成功！                    │
│  共計算 350 個欄位。                  │
│                                     │
│           [查看結果]  [關閉]        │
│                                     │
└─────────────────────────────────────┘
```

**特殊屬性**：
- 標題左側顯示成功圖標（綠色）
- 標題文字：`#52c41a`

## 8. 加載狀態 (Loading)

### 8.1 Spinner

**樣式規格**：
```
    ⟳
```

| 屬性 | 值 |
|------|------|
| 直徑 | `32px` (默認) / `48px` (大) |
| 顏色 | `#1890ff` |
| 動畫 | 順時針旋轉，1 秒/圈 |

---

### 8.2 按鈕加載

**樣式規格**：
```
┌─────────────────┐
│  ⟳  計算中...   │
└─────────────────┘
```

**特殊屬性**：
- Spinner 在左側
- 按鈕禁用（防止重複提交）

---

### 8.3 頁面加載
