# 年結表計算器 - UI 設計規範

> 本文檔專注於**用戶界面設計**，包含頁面佈局、視覺規範、交互設計、組件庫等。

---

## 🎨 設計原則

1. **減少點擊** —— 優先使用單選按鈕，減少下拉選單
2. **清晰分組** —— 相關字段放在同一卡片內
3. **即時反饋** —— 輸入錯誤時立即提示
4. **一致性** —— 與現有 LIS7 POS52 年結表模板保持視覺一致

## 🧩 組件庫

### 按鈕

| 類型 | 樣式 | 使用場景 |
|------|------|----------|
| 主按鈕 | 藍色背景 `#1890ff`，白字 | 計算、提交、保存 |
| 次要按鈕 | 白色背景，灰色邊框 | 重置、取消、返回 |
| 危險按鈕 | 紅色背景 `#f5222d`，白字 | 刪除、停用 |
| 鏈接按鈕 | 無背景，藍色文字 | 展開詳情、查看更多 |

### 輸入框

| 類型 | 樣式 | 說明 |
|------|------|------|
| 默認 | 白色背景，灰色邊框 | 正常狀態 |
| 聚焦 | 藍色邊框 `#1890ff` | 用戶輸入時 |
| 錯誤 | 紅色邊框 `#f5222d` | 驗證失敗 |
| 禁用 | 淺灰背景 `#f5f5f5` | 不可編輯 |

### 單選按鈕

- 橫向排列（減少點擊）
- 選中：藍色填充
- 未選中：灰色圓圈

### 卡片

- 白色背景
- 淺灰邊框 `#d9d9d9`
- 圓角 `8px`
- 內邊距 `24px`
- 陰影（可選）

### 表格

- 表頭：淺灰背景 `#fafafa`
- 交替行：白色 / 淺灰
- 邊框：淺灰 `#d9d9d9`
- 懸停行：淺藍背景 `#e6f7ff`

## ✨ 交互規範

### 表單驗證

| 場景 | 觸發時機 | 提示方式 |
|------|----------|----------|
| 必填字段 | 失去焦點時 | 紅色邊框 + 下方紅色文字 |
| 日期格式 | 失去焦點時 | 紅色邊框 +「日期格式無效」 |
| 年齡範圍 | 失去焦點時 | 紅色邊框 +「年齡必須在 18-80 之間」 |
| 金額格式 | 失去焦點時 | 紅色邊框 +「請輸入有效數字」 |

### 加載狀態

- 計算中：顯示 spinning loader
- 超過 2 秒：顯示「計算中，請稍候...」
- 禁止重複提交（按鈕禁用）

### 錯誤提示

- 輕微錯誤：輸入框下方紅色文字
- 嚴重錯誤：彈窗（居中，帶遮罩）
- 網絡錯誤：頂部通知條（可關閉）

### 成功提示

- 輕微成功：右上角綠色通知（3 秒自動消失）
- 重要成功：彈窗確認（如「計算完成」）

## 📱 響應式設計

**目標設備**：桌面端（內部系統，不支持移動設備）

| 屏幕寬度 | 佈局策略 |
|----------|----------|
| ≥1440px | 標準佈局，最大寬度 1440px |
| 1024px-1439px | 自適應縮放 |
| <1024px | 表格橫向滾動，卡片堆疊 |

## 🎯 無障礙設計

- 所有輸入框有明確 label
- 錯誤提示使用顏色 + 文字（不只靠顏色）
- 鍵盤可操作（Tab 鍵導航）
- 對比度符合 WCAG 2.0 AA 標準
