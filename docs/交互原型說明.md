## 📱 P01: 表單填寫流程

### 路徑
```
首頁 → 點擊「快速查詢」→ 查詢輸入頁
```

### 交互步驟

#### 步驟 1：進入頁面
```
用戶動作：點擊「快速查詢」按鈕
系統反應：
  - 頁面跳轉至查詢輸入頁
  - 自動聚焦到第一個字段（退保日期）
  - 所有字段為空，默認狀態
```

---

#### 步驟 2：選擇產品
```
用戶動作：點擊產品選擇器
系統反應：
  - 彈出下拉選單
  - 顯示所有產品列表
  - 每個產品顯示：名稱 + 代碼

用戶動作：選擇「『世』心終身壽險計劃」
系統反應：
  - 關閉下拉選單
  - 顯示選中的產品
  - 自動聚焦到下一個字段（保單生效日）
```

---

#### 步驟 3：輸入日期
```
用戶動作：點擊日期輸入框
系統反應：
  - 彈出日期選擇器
  - 顯示當前月份日曆
  - 可選擇年/月切換

用戶動作：選擇日期（2026-03-24）
系統反應：
  - 關閉日期選擇器
  - 輸入框顯示「2026-03-24」
  - 自動格式化
  - 自動聚焦到下一個字段
```

---

#### 步驟 4：選擇性別
```
用戶動作：點擊「男」單選按鈕
系統反應：
  - 「男」按鈕變為選中狀態（藍色填充）
  - 「女」按鈕變為未選中狀態
  - 無延遲，即時反饋
```

---

#### 步驟 5：輸入金額
```
用戶動作：輸入數字（123456.78）
系統反應：
  - 即時顯示輸入內容
  - 自動添加千分位分隔符：「123,456.78」
  - 限制小數點後 2 位

用戶動作：輸入非數字字符
系統反應：
  - 阻止輸入
  - 或自動過濾
```

---

#### 步驟 6：展開「其他」區域
```
用戶動作：點擊「▶ 展開」按鈕
系統反應：
  - 按鈕圖標旋轉為「▼」
  - 區域向下展開（動畫 0.3s）
  - 顯示附加字段

用戶動作：點擊「▼ 收起」按鈕
系統反應：
  - 按鈕圖標旋轉為「▶」
  - 區域向上收起（動畫 0.3s）
  - 隱藏附加字段
```

## ✅ P02: 表單驗證流程

### 路徑
```
查詢輸入頁 → 輸入無效數據 → 失去焦點 → 顯示錯誤
```

### 交互步驟

#### 步驟 1：必填字段為空
```
用戶動作：跳過必填字段，點擊下一個字段
系統反應：
  - 當前字段邊框變紅（#f5222d）
  - 字段下方顯示錯誤提示
  - 錯誤圖標（⚠️）顯示

錯誤提示示例：
  「退保日期為必填」
```

---

#### 步驟 2：日期格式錯誤
```
用戶動作：輸入無效日期（2026-13-45）
系統反應：
  - 失去焦點時驗證
  - 邊框變紅
  - 顯示錯誤提示

錯誤提示：
  「日期格式無效」

用戶動作：修正為有效日期（2026-03-24）
系統反應：
  - 失去焦點時驗證
  - 邊框恢復正常（#d9d9d9）
  - 錯誤提示消失
```

---

#### 步驟 3：年齡超出範圍
```
用戶動作：輸入年齡（150）
系統反應：
  - 失去焦點時驗證
  - 邊框變紅
  - 顯示錯誤提示

錯誤提示：
  「年齡必須在 0-120 之間」

用戶動作：修正為有效年齡（35）
系統反應：
  - 邊框恢復正常
  - 錯誤提示消失
```

---

#### 步驟 4：字段聯動驗證
```
用戶動作：
  1. 輸入保單生效日：2026-03-24
  2. 輸入繳至日期：2025-01-01

系統反應：
  - 繳至日期失去焦點時驗證
  - 邊框變紅
  - 顯示錯誤提示

錯誤提示：
  「繳至日期必須晚於保單生效日」

用戶動作：修正繳至日期為 2027-01-01
系統反應：
  - 邊框恢復正常
  - 錯誤提示消失
```

---

#### 步驟 5：提交前驗證
```
用戶動作：點擊「計算」按鈕（有必填字段為空）
系統反應：
  - 不提交表單
  - 所有必填字段驗證
  - 第一個錯誤字段自動聚焦
  - 顯示錯誤提示

用戶動作：修正所有錯誤
系統反應：
  - 所有錯誤提示消失
  - 進入計算流程
```

## ⏳ P03: 計算加載流程

### 路徑
```
查詢輸入頁 → 點擊「計算」→ 加載中 → 成功/失敗
```

### 交互步驟

#### 步驟 1：點擊計算按鈕
```
用戶動作：點擊「計算」按鈕
系統反應：
  - 按鈕文字變為「計算中...」
  - 按鈕左側顯示 spinner
  - 按鈕禁用（防止重複提交）
  - 所有輸入框禁用
```

---

#### 步驟 2：計算中（0-2 秒）
```
系統狀態：
  - 按鈕保持「計算中...」狀態
  - Spinner 持續旋轉
  - 用戶等待
```

---

#### 步驟 3：計算中（超過 2 秒）
```
系統反應：
  - 顯示彈窗
  - 彈窗內容：
    ┌─────────────────────────────────┐
    │                                 │
    │            ⟳                   │
    │       計算中，請稍候...          │
    │      正在計算 350 個欄位          │
    │                                 │
    └─────────────────────────────────┘
  - 彈窗無關閉按鈕
  - 背景遮罩（不可點擊）
```

---

#### 步驟 4a：計算成功
```
系統反應：
  - 關閉加載彈窗（如有）
  - 按鈕恢復正常狀態
  - 右上角顯示成功通知：
    ┌─────────────────────────────────┐
    │  ✅  計算成功                    │
    └─────────────────────────────────┘
  - 通知 3 秒後自動消失
  - 1 秒後自動跳轉至結果展示頁
```

---

#### 步驟 4b：計算失敗
```
系統反應：
  - 關閉加載彈窗（如有）
  - 按鈕恢復正常狀態
  - 所有輸入框啟用
  - 顯示錯誤彈窗：
    ┌─────────────────────────────────┐
    │  ⚠️  計算失敗                    │
    ├─────────────────────────────────┤
    │                                 │
    │  所選產品的年齡無效。            │
    │  請確認年齡在 18-80 歲之間。      │
    │                                 │
    │                [確定]           │
    │                                 │
    └─────────────────────────────────┘

用戶動作：點擊「確定」
系統反應：
  - 關閉彈窗
  - 自動聚焦到錯誤字段（年齡）
  - 年齡字段邊框變紅
```

## 📄 P04: 結果展示與導出

### 路徑
```
查詢輸入頁 → 計算成功 → 結果展示頁 → 導出/列印/郵件
```

### 交互步驟

#### 步驟 1：進入結果頁
```
系統反應：
  - 頁面從右側滑入（動畫 0.3s）
  - 顯示保單資料表格
  - 顯示保單戶口價值表格
  - 顯示說明摘要
```

---

#### 步驟 2：導出 PDF
```
用戶動作：點擊「📄 導出 PDF」按鈕
系統反應：
  - 按鈕變為「計算中...」狀態
  - 顯示加載提示
  - 生成 PDF 文件
  - 自動下載到本地

下載完成：
  - 按鈕恢復正常
  - 右上角顯示成功通知：
    「PDF 已生成並下載」
```

---

#### 步驟 3：列印
```
用戶動作：點擊「🖨️ 列印」按鈕
系統反應：
  - 調用瀏覽器列印對話框
  - 使用專用列印樣式（黑白，無背景色）
```

---

#### 步驟 4：發送郵件
```
用戶動作：點擊「✉️ 發送郵件」按鈕
系統反應：
  - 顯示彈窗：
    ┌─────────────────────────────────┐
    │  發送年結表                      │
    ├─────────────────────────────────┤
    │                                 │
    │  收件人郵箱：                    │
    │  [__________________________]   │
    │                                 │
    │  郵件內容：                      │
    │  [__________________________]   │
    │  [__________________________]   │
    │                                 │
    │           [取消]  [發送]        │
    │                                 │
    └─────────────────────────────────┘

用戶動作：輸入郵箱，點擊「發送」
系統反應：
  - 顯示加載提示
  - 發送成功後顯示通知
  - 關閉彈窗
```

---

#### 步驟 5：重新計算
```
用戶動作：點擊「🔄 重新計算」按鈕
系統反應：
  - 跳轉回查詢輸入頁
  - 保留所有已輸入的字段
  - 用戶可修改部分字段後重新計算
```

## 🏠 P05: 首頁快速導航

### 路徑
```
登錄後 → 首頁
```

### 交互步驟

#### 步驟 1：點擊「快速查詢」
```
用戶動作：點擊「快速查詢」大按鈕
系統反應：
  - 按鈕有點擊效果（陰影變化）
  - 頁面跳轉至查詢輸入頁
```

---

#### 步驟 2：點擊「管理後台」（僅精算管理員）
```
用戶動作：點擊「管理後台」按鈕
系統反應：
  - 按鈕有點擊效果
  - 頁面跳轉至管理後台頁

非精算管理員：
  - 按鈕隱藏或禁用
  - 鼠標懸停提示：「無權限訪問」
```

---

#### 步驟 3：點擊最近查詢記錄
```
用戶動作：點擊某條最近查詢記錄
系統反應：
  - 跳轉至結果展示頁
  - 顯示該次查詢的結果
```

## 📤 P06: 費率表上傳（管理後台）

### 路徑
```
管理後台 → 費率表管理 → 上傳 Excel
```

### 交互步驟

#### 步驟 1：導出模板
```
用戶動作：點擊「導出模板」按鈕
系統反應：
  - 顯示加載提示
  - 下載 Excel 模板文件
  - 通知：「模板已下載」
```

---

#### 步驟 2：上傳文件
```
用戶動作：點擊「上傳文件」按鈕
系統反應：
  - 打開文件選擇對話框
  - 僅接受 .xlsx 文件

用戶動作：選擇文件
系統反應：
  - 顯示文件名稱
  - 顯示文件大小
  - 顯示「預覽」按鈕
```

---

#### 步驟 3：預覽
```
用戶動作：點擊「預覽」按鈕
系統反應：
  - 顯示彈窗
  - 表格形式顯示前 10 行數據
  - 顯示總行數
  - 顯示驗證結果

驗證通過：
  ┌─────────────────────────────────┐
  │  預覽                            │
  ├─────────────────────────────────┤
  │  ✅ 格式正確，共 1234 行          │
  │                                 │
  │  [取消]  [確認上傳]              │
  └─────────────────────────────────┘

驗證失敗：
  ┌─────────────────────────────────┐
  │  預覽                            │
  ├─────────────────────────────────┤
  │  ⚠️ 發現 3 個錯誤                  │
  │  - 第 5 行：年齡無效              │
  │  - 第 12 行：費率格式錯誤         │
  │  - 第 89 行：缺少必填字段        │
  │                                 │
  │  [取消]  [下載錯誤報告]          │
  └─────────────────────────────────┘
```

---

#### 步驟 4：確認上傳
```
用戶動作：點擊「確認上傳」
系統反應：
  - 顯示加載提示
  - 上傳進度條：
    ┌─────────────────────────────┐
    │  ████████████░░░░░  65%     │
    └─────────────────────────────┘
  - 上傳完成後顯示成功通知
  - 自動刷新費率表列表
```

## 🎞️ 動畫規格

### 頁面切換
| 類型 | 動畫 | 時長 |
|------|------|------|
| 進入頁面 | 從右側滑入 | 0.3s |
| 離開頁面 | 從左側滑出 | 0.3s |
| 彈窗出現 | 淡入 + 放大 | 0.2s |
| 彈窗關閉 | 淡出 + 縮小 | 0.2s |

### 表單交互
| 類型 | 動畫 | 時長 |
|------|------|------|
| 字段聚焦 | 邊框顏色過渡 | 0.2s |
| 錯誤提示 | 從上滑入 | 0.2s |
| 卡片展開 | 向下展開 | 0.3s |
| 卡片收起 | 向上收起 | 0.3s |

### 按鈕狀態
| 類型 | 動畫 | 時長 |
|------|------|------|
| Hover | 背景色過渡 | 0.2s |
| Click | 縮放 0.95 | 0.1s |
| Disabled | 透明度過渡 | 0.2s |
