## 📋 IF01: 查詢年結表流程

### 流程圖
```
首頁
  ↓ [點擊快速查詢]
查詢輸入頁
  ↓ [填寫表單]
  ↓ [點擊計算]
驗證表單
  ├─→ [驗證失敗] → 顯示錯誤 → 修正表單
  └─→ [驗證通過] → 調用計算 API
        ├─→ [計算失敗] → 顯示錯誤彈窗 → 返回表單
        └─→ [計算成功] → 結果展示頁
```

### 詳細步驟

#### 步驟 1：進入查詢頁
```
起點：首頁
觸發：用戶點擊「快速查詢」按鈕
動作：
  - 頁面跳轉至查詢輸入頁
  - 自動聚焦到第一個字段（退保日期）
  - 表單為空，所有字段為默認狀態
終點：查詢輸入頁（空表單）
```

#### 步驟 2：填寫表單
```
觸發：用戶輸入字段
動作：
  - 實時顯示輸入內容
  - 失去焦點時觸發驗證
  - 驗證通過：顯示綠色對勾（可選）
  - 驗證失敗：顯示紅色邊框 + 錯誤提示
終點：表單填寫完成
```

#### 步驟 3：點擊計算
```
觸發：用戶點擊「計算」按鈕
前置條件：所有必填字段已填寫
動作：
  - 按鈕變為加載狀態（「計算中...」+ spinner）
  - 按鈕禁用（防止重複提交）
  - 所有輸入框禁用
  - 調用計算 API
終點：等待 API 響應
```

#### 步驟 4a：驗證失敗
```
觸發：API 返回驗證錯誤
動作：
  - 按鈕恢復正常狀態
  - 所有輸入框啟用
  - 顯示錯誤彈窗
  - 自動聚焦到第一個錯誤字段
  - 錯誤字段顯示紅色邊框
終點：用戶修正表單
```

**錯誤彈窗內容**：
```
┌─────────────────────────────────────┐
│  ⚠️  計算失敗                        │
├─────────────────────────────────────┤
│                                     │
│  所選產品的年齡無效。                │
│  請確認年齡在 18-80 歲之間。          │
│                                     │
│                  [確定]             │
│                                     │
└─────────────────────────────────────┘
```

---

#### 步驟 4b：計算成功
```
觸發：API 返回計算結果
動作：
  - 按鈕恢復正常狀態
  - 顯示成功通知（右上角，3 秒）
  - 1 秒後自動跳轉至結果展示頁
  - 攜帶計算結果數據
終點：結果展示頁
```

**成功通知**：
```
┌─────────────────────────────────────┐
│  ✅  計算成功                        │
└─────────────────────────────────────┘
```

## 📋 IF02: 表單驗證流程

### 流程圖
```
用戶輸入字段
  ↓
失去焦點
  ↓
觸發驗證
  ├─→ 必填驗證
  ├─→ 格式驗證
  ├─→ 範圍驗證
  └─→ 聯動驗證
        ├─→ [驗證失敗] → 顯示錯誤
        └─→ [驗證通過] → 清除錯誤
```

### 驗證規則

#### 必填驗證
```
字段：退保日期、繳至日期、產品名稱、保單生效日、年齡、性別、吸煙狀態、投保金額、已繳保費
規則：不為空
錯誤提示：「[字段名] 為必填」
```

#### 格式驗證
```
字段：所有日期字段
規則：YYYY-MM-DD 格式，有效日期
錯誤提示：「日期格式無效」

字段：年齡、金額
規則：數字格式
錯誤提示：「請輸入有效數字」
```

#### 範圍驗證
```
字段：年齡
規則：0-120
錯誤提示：「年齡必須在 0-120 之間」

字段：投保金額
規則：>0
錯誤提示：「投保金額必須大於 0」

字段：已繳保費
規則：≥0
錯誤提示：「已繳保費必須大於等於 0」
```

#### 聯動驗證
```
規則 1：繳至日期 ≥ 保單生效日
錯誤提示：「繳至日期必須晚於保單生效日」

規則 2：退保日期 ≥ 保單生效日
錯誤提示：「退保日期必須晚於保單生效日」

規則 3：年齡與保單生效日匹配
錯誤提示：「年齡與保單生效日不匹配」
```

## 📋 IF06: 上傳費率表流程

### 流程圖
```
管理後台 - 費率表管理
  ↓ [點擊上傳費率表]
選擇費率表類型
  ↓ [選擇文件]
上傳文件
  ↓ [驗證格式]
  ├─→ [驗證失敗] → 顯示錯誤報告
  └─→ [驗證通過] → 顯示預覽
        ↓ [確認上傳]
      上傳 API
        ├─→ [上傳失敗] → 顯示錯誤
        └─→ [上傳成功] → 顯示結果 + 刷新
```

### 詳細步驟

```
觸發：用戶點擊「📤 上傳費率表」按鈕

  - 顯示第一步彈窗
  - 選擇費率表類型（DB/AD/TD/Prem/CSV/COU）

```

動作：
  - 驗證文件格式（.xlsx）
  - 驗證文件大小（<10MB）
  - 顯示文件名
  - 「下一步」按鈕啟用
終點：文件選擇完成
```

---

#### 步驟 3：驗證格式
```
觸發：用戶點擊「下一步」按鈕
動作：

  - 調用驗證 API

## 📋 IF08: 數據備份流程

### 流程圖
```
系統管理 - 數據備份
  ↓ [點擊立即備份]
顯示確認彈窗
  ↓ [點擊開始備份]
顯示進度彈窗
  ├─→ [用戶取消] → 確認取消 → 停止備份
  └─→ [備份完成] → 顯示結果
        ├─→ [備份失敗] → 顯示錯誤
        └─→ [備份成功] → 關閉彈窗 + 刷新
```

### 詳細步驟

動作：

  - 可選填寫備註
終點：用戶確認
```

---

#### 步驟 2：開始備份
```
觸發：用戶點擊「開始備份」按鈕
動作：
  - 關閉確認彈窗
  - 顯示進度彈窗
  - 調用備份 API
  - 開始定時更新進度
終點：等待備份完成
```

---

```
觸發：定時器（每秒）
動作：

## 🎯 異常流程處理

### 網絡錯誤
```
觸發：API 調用超時或失敗
處理：
  - 顯示網絡錯誤通知
  - 提供重試按鈕
  - 記錄錯誤日誌
```

### 服務器錯誤
```
觸發：API 返回 5xx 錯誤
處理：
  - 顯示服務器錯誤通知
  - 建議稍後重試
  - 記錄錯誤日誌
  - 通知管理員
```

### 權限錯誤
```
觸發：API 返回 403 錯誤
處理：
  - 顯示權限錯誤通知
  - 禁用相關按鈕
  - 建議聯繫管理員
```

### 數據衝突
```
觸發：API 返回 409 錯誤（如：代碼重複）
處理：
  - 顯示衝突錯誤
  - 標記衝突字段
  - 建議修改後重試
```
